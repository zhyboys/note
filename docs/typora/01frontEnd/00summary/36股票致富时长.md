# 36 股票致富时长

```js
const 百 = 100, 千 = 1000, 万 = 10000, 十万 = 100000, 百万 = 1000000, 千万 = 10000000, 亿 = 100000000
let 本金 = 万
let 目标 = 千万
let 日涨幅 = 0.01



获取发财要多久(本金, 目标, 日涨幅)

function 获取发财要多久(本金, 目标, 日涨幅) {
    // 公式：本金 * (1.1^天数) = 目标
    日涨幅 = 日涨幅 + 1
    const 交易日 = Math.log(目标 / 本金) / Math.log(日涨幅)
    const 交易月 = 交易日 / (5 * 4)
    const 交易年 = 交易月 / 12
    console.log(`\n\n${交易日.toFixed(1)}个交易日，${交易月.toFixed(1)}个交易月，${交易年.toFixed(1)}个交易年\n`);
}
```
