# 06 混合

混合就是将一系列属性从一个规则集引入到另一个规则集的方式。

```html
<div class="father">
    <div class="son"></div>
</div>
```

```less
/*
2.less中混合的注意点:
如果混合名称的后面没有(), 那么在预处理的时候, 会保留混合的代码
如果混合名称的后面加上(), 那么在预处理的时候, 不会保留混合的代码
 */
/*
.center{
  position: absolute;
  left: 50%;
  top: 50%;
  transform: translate(-50%, -50%);
}
*/
.center(){
  position: absolute;
  left: 50%;
  top: 50%;
  transform: translate(-50%, -50%);
}
/*
1.什么是less中的混合(Mix in)?
将需要重复使用的代码封装到一个类中, 在需要使用的地方调用封装好的类即可
在预处理的时候less会自动将用到的封装好的类中的代码拷贝过来
本质就是ctrl+c  --> ctrl + v
 */
.father{
  width: 300px;
  height: 300px;
  background: red;
  .center();//使用时可以加（），也可以不加
  .son{
    width: 200px;
    height: 200px;
    background: blue;
    .center();
  }
}
```
